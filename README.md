# 🔍 Lost & Found Portal

A web-based portal designed for colleges and societies to help students and members report and find lost items efficiently.

## 🎯 Problem Statement

In colleges and societies, people frequently lose items like ID cards, wallets, keys, and books. There's no centralized system to report or search for these items, making it difficult to reunite lost items with their owners.

## 💡 Solution

This Lost & Found Portal provides a simple platform where users can:
- Post items they have found
- Report items they have lost
- Search and filter through all posted items
- Contact the person who posted the item directly

## ✨ Features

- **Post Items**: Users can post lost or found items with detailed descriptions
- **Color-Coded Display**: 
  - 🔴 Lost items displayed in RED
  - 🟢 Found items displayed in GREEN
- **Smart Filtering**: Filter items by All/Lost/Found with one click
- **Contact Information**: Each post includes email and phone number for easy communication
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Data Persistence**: Uses LocalStorage to save data even after browser refresh
- **Delete Option**: Users can remove their posts when items are found/returned

## 🛠️ Tech Stack

- **HTML5** - Structure and semantic markup
- **CSS3** - Styling with gradients, flexbox, and grid layout
- **JavaScript (ES6)** - Core functionality and DOM manipulation
- **LocalStorage API** - Client-side data persistence

## 💻 Usage Guide

1. **Posting an Item**:
   - Select item type (Lost/Found)
   - Enter item name, location, and description
   - Provide your contact details
   - Click "Post Item"

2. **Finding Items**:
   - Browse all items in the list
   - Use filter buttons to view only Lost or Found items
   - Click on email/phone to contact the poster

3. **Deleting Items**:
   - Click the "Delete" button on your post when item is recovered
